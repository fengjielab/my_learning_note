#  从零开始：GitHub 克隆 + UV 虚拟环境搭建指南

## 第一步：安装 UV

macOS / Linux

curl -LsSf https://astral.sh/uv/install.sh | sh
uv --version

## 第二步：克隆项目
替换为实际的 GitHub 仓库地址
git clone https://github.com/username/repository.git

进入项目目录
cd repository
## 第三步：创建虚拟环境（默认 Python 版本）
uv venv

或指定 Python 版本（UV 会自动下载）
uv venv --python 3.11

查看创建的虚拟环境
ls -la .venv  # Linux/macOS
## 第四步安装项目及其所有依赖，并生成/更新 uv.lock 锁定文件
uv pip install -r requirements.txt
uv sync
## 第五步激活虚拟环境
source .venv/bin/activate
python main.py
